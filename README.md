# Netflix-Data-Cleaning
Data cleaning and pre processing of Netflix dataset using Excel

## Dataset
Netflix Movies and TV Shows dataset from Kaggle.
## Objective
To clean and preprocess the dataset by handling missing values, standardizing text fields, removing inconsistencies, and exporting cleaned data for analysis.

## Tools Used
- Microsoft Excel

## Cleaning Steps
- Identified missing values
- Checked for duplicates
- Standardized text using TRIM, PROPER, and UPPER
- Validated date and categorical formats
- Created cleaned dataset and quality notes

## Output Files
- Cleaned_dataset.xlsx
- cleaned_dataset.csv
